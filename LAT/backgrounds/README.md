
# background images

change background: url with your image file name in myCustom.css

```css
.content-wrapper {
    min-height: 100%;
    background-color: #ecf0f5;
    background: url(background/slide_2.jpg);
    background-repeat: no-repeat;
    background-attachment: fixed;
    background-size: cover;
    z-index: 800;
  }
  ```
